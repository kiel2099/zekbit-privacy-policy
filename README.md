# zekbit-privacy-policy
Privacy policy for the Zekbit Android app

Privacy Policy

Zekbit is a paid web browser application that respects user privacy.

Data Collection and Usage:
Zekbit does not collect, store, or share personal user data such as name, email address, or location.

Browsing Data:
The app provides features such as bookmarks (favorites) and browsing history. This information is stored locally on the user's device and is not transmitted to any external servers.

Internet Access:
Zekbit accesses the internet solely to load and display web content requested by the user. The app does not monitor, track, or analyze browsing activity.

Data Sharing:
Zekbit does not share any user data with third parties.

Security:
All user data (such as bookmarks and browsing history) remains on the device and is under the user’s control.

Changes to This Policy:
This policy may be updated in the future. Any changes will be reflected on this page.
